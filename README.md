# B1 Coach

A proof-of-concept AI coaching layer for TrackMan B1 baseball hitting data.

TrackMan already does data collection and visualization well. B1 Coach is the interpretation layer — it takes per-session hitting metrics and delivers them the way a coach would after batting practice: a few specific observations grounded in the actual numbers, two concrete tips for next session, and a conversational AI coach the player can ask follow-up questions.

## What it does

- Player selects a goal before each session (Power, Line Drives, Contact, Reduce Pop-Ups, Hit to All Fields, or Open Session)
- Live session screen animates incoming swing data in real time
- Post-session debrief is generated by Claude using session metrics — exit velocity, launch angle, pitch location, spray direction, and distance
- Two charts are selected by Claude based on the goal and session data patterns
- Virtual Coach chat allows the player to ask follow-up questions with full session context
- Up to four sessions per practice run, with prior session history passed to Claude for coaching continuity

## Tech stack

React, Vite, Tailwind CSS, Anthropic API (Claude Sonnet), Vercel serverless functions

The Anthropic API key is protected server-side via a Vercel serverless function. The browser never sees the key.

## Status

Proof-of-concept running on mock data structured to match the real TrackMan B1 API. Real API integration is a defined next step.
